연산자

산술 연산자

        +, -, *,/ 에 대해 알아 볼 것
        % 란 ?   나눈 나머지이다 나누기랑 다른 것이 
        나누기는 나눈 값이지만 %는 말그대로 사칙연산에서 나누고 남은 나머지이다

문자열 더하기

    '문자' + 20  ->  문자20
    '문자' + 20 + 30 ->  문자2030
    20+30+'문자'  -> 50문자
    10+'20'  ->      1020
    10 -'20' -> - 10

증감 연산자

       i++ 란? 실행된뒤에 값을 1씩 추가
       ++i 란? 실행되자마자 1씩추가

복합대입연산자

        + = 더한 결과
        - = 곱합 결과
        * = 나눈 결과
        / = 나눈나머지 결과
