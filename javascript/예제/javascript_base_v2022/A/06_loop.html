Loop

for 반복문

        for(초기값;조건;증감){
            반복...
        }

while 반복문

        초기값;
        while(조건){
            반복할 일
            증감;
        }

do while 반복문

        초기값;
        do{
            반복할일;
            증감;
        } while(조건);

forEach 반복문

        배열의 값으로 반복할 때
        대상.forEach(funtion(원소,인덱스,모든값){

        });

for in 반복문

        객체의 값 반복
        for(원소 in 객체명){
            반복할 일
        }

for of 반복문

        배열, index 번호 사용x 

        for(원소 of 배열명){
            반복할 일
        }
