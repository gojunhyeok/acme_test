Array

        새배열 생성 
        var 배열명 = []; //빈 배열 생성

        배열에 값을 추가
        배열명[인덱스번호] = 값 // index 0부터 시작

length

        배열의 개수 length
        배열명.length

join

        배열의 값 병합 -> 문자열
        배열명.join();
        배열명.join('/');     - _ 등등을 넣으면 된다

reverse

        배열의 값(원소)을 역순으로. 주의할점은 원본이 수정된다.

sort

        내림, 오름, 알파벳, 가나다.... - 배열의 값을 정렬  주의점으로 원본이 수정됨
        대상.sort() // 영어면 알파벳순 숫자면 오림차순으로 해줌
        return 으로 -1의 값을 받으면 역순으로 정렬된다.

push

        배열의 마지막에 값 추가

pop

        배열의 마지막 값을 제거

unshift

        배열의 앞의 값을 추가

shift

        배열의 앞의 값을 제거

slice

        인덱스번호 해당 원소를 복사 추출
        대상.slice(시작, 끝); 끝번호 앞까지

splice

        인덱스번호 해당 원소를 오려내어 추출
        대상.splice(시작, 끝); 끝번호 앞까지

concat

        대상.concat(합칠 대상) 합칠 대상-배열명, 값

filter

        조건에 맍는 값들로 새 배열을 생성
        대상.filter(function(매개변수)){
            return 매개변수...
        }
