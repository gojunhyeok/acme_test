으로 시작한다.
여기서 메서드부분의 post는 아래항목에서 서술하겠다.


form 의 메서드에는 get과 post 가 있다.


    get
 암호화 x, 주소표시줄에 그대로 노출된다. 긴 값 전달 이 불가능 하다.

 get 방식으로 보낸 form 의 데이터는 주소창에 데이터파라미터 그대로의 내용이 브라우저 또는 ui 
 에 그대로 표시된다.

ex) 네이버 검색창에 아무거나 검색할시 주소창이 암호화되지않고 상단에 그대로 노출됨

    post 
암호화 o html 태그의 body 안에 숨어서 긴 값 전달이 가능하다.
이것은 head 태그가아닌 body 태그 안에서의 데이터간의 교류이기때문에 
따로 주소창같은 ui 에서 데이터 파라미터가 이동하는 것을 유저가 볼수는 없다.


별개로 ★https 와 헷갈리는 경우가 있는데★

https 의 암호화는
form 태그의 get 방식이든 post 이든 그것을 전체적으로 암호화 해서 보내는 것의 
웹페이ㅣ의 https 의 방식이고

http 는 body 로 보내든 get 으로보내든 그전체의 form 데이터를 암호화 없이 보내는 것의

차이가 있따.









    method="get" 

input 태그에서 method="get"  를 사용할 경우 사용자가 어떠한 데이터를 넣어도
서버에서 즉각 데이터를 리턴 해준다. (확실x)

    method="post" 
input 태그에서 method="post" 를 사용할시 암호화된 입력을 작성해야 서버에서
데이터를 리턴해준다 (확실 x)



입력이 반드시 요구된다는 속성을 넣을떈

필수 입력 사항
    required="required" (중요해)

를 사용하여 알맞은 값을 넣을때만 form 이 작동하도록 하여야 한다.

ex) 주민등록번호 성별 전화번호 등등



    ★ form 을 사용하기위한 input 의 요소들 ★

    placeholder 요소

텍스트 입력란에 힌트문구를 추가하여 사용자가 어떠한 텍스트를 입력해야 하는지 알려준다.

    value 요소

input 에서의 value 요소는 입력되는 값을 내가 직접 지정해 놓는 값이다.


    minlength, maxlength 요소


minlength 란?

minlength 를 추가할경우 이용자가 적을 수 있는 최소텍스트의 량이 조절된다.

ex)minlength="5" 를 할경우 최소 적어야 하는 텍스트 입력값은 5가됨


maxlength 란?

maxlength 를 추가할경우 이용자가 적을 수 있는 최대텍스트의 량이 조절된다.

ex)mexlength="5" 를 할경우 최대 적어 수 있는 텍스트는 5가됨



     size 요소

size 요소를 추가할경우 입력칸의 사이즈(너비)가 조절됨
그러나 이것의 요소를 추가하는 것은 바람직 하지않다. 
css 에서 입력칸을 따로 조절할 것 이기 때문


    readonly 요소

readonly  요소를 추가할경우 텍스트 입력칸에 텍스트를 작성할 수 없게 된다.

    novalidate 요소

 기본적인 브라우저에선 유효성 검사가 진행되지만 form 태그에 novalidate 요소를 추가할경우 
브라우저의 유효성 검사가 기본으로 실행이 안된다.
이 것을 하는 이유는 브라우저의 유효성 검사가아닌 내가 직접 코드를 작성해 유효성 검사를 할경우
이 요소를 추가하여 기본적인 브라우저의 유효성 검사를 막아 놔야 하기 때문이다.

     type="tel"

사용자의 전화번호 등을 물을때 사용하는 요소 

    input type="reset"  

인풋태그에 추가할경우 사용자가 적어놓은 텍스트를 초기화 시키는 이다.
(사용자 편의차원에서 입력대신 실수로 누를 수 있기때문에 실무에서는 잘 사용안하는 편)

    type="file"

파일 요소를 사용할 경우 pc내의 파일을 웹사이트에 등록 시키는 버튼을 추가할 수 있다.
태그요소에 추가로 

    ★"multiple"★ (중요)
     요소를 추가할시 파일의 다중선택이 가능하다.


     type="url"

홈페이지 주소 url 을 입력해야 하는 입력란이 생성된다.
이곳에는 url 만 들어갈 수 있다.

    type="number" "min="0" max="10" step="2"

경력이나 아파트층 등 원하는 숫자의 선택지를 고르는 창이 생성된다.
여기서 min 은 최소값 max 는 최대값 step 목록에서 넘어가는 숫자의 수

    type="range"

범위 조절이가능한 게이지바 를 생성한다.
ex) 포토샵의 오퍼시티 게이지바

    type="color"
사용자가 웹페이지에서 컬러를 선택할 수 있는 rgb 창이 생성된다.

    type="search"
웹페이제란에 검색란이 추가된다.
